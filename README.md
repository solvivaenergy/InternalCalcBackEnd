# InternalCalcBackEnd

Render-ready backend for the Solviva calculator.

## Endpoints

- `GET /health`
- `POST /api/quote`

## Environment variables

- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `PORT` (optional, default `3000`)
- `CORS_ORIGINS` (optional, comma-separated list or `*`)

## Local run

```bash
npm install
npm start
```
